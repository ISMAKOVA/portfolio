# Personal Website

## Core technologies: 
### base:
- [next.js 13+](https://nextjs.org/)
- [typescript](https://www.typescriptlang.org/)
### ui:
- [tailwind css](https://tailwindcss.com/)
- [lucide](https://lucide.dev/)
- [framer motion](https://www.framer.com/motion/)


## Getting Started

Run to start development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
